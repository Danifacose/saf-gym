# 📋 Directives

Questa cartella contiene le **SOP (Standard Operating Procedures)** del progetto PEPPESITO.

## Cos'è una direttiva?

Una direttiva è un file Markdown che descrive **come eseguire un task specifico**. Contiene:

- **Obiettivo**: cosa deve essere fatto
- **Input**: quali dati/informazioni servono
- **Tool/Script**: quali script in `execution/` usare
- **Output**: cosa ci si aspetta come risultato
- **Casi limite**: errori noti, edge case, workaround

## Come creare una nuova direttiva

Crea un file `.md` con un nome descrittivo, ad esempio:
- `scrape_website.md`
- `generate_report.md`
- `sync_google_sheets.md`

## Principio

Le direttive sono **documenti vivi**: vanno aggiornate ogni volta che si scoprono nuovi vincoli, errori o approcci migliori.
